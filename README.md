# Система контроля территории
## 1. Запуск основного приложения в модуле main.
## 2. Разметка и cохранение координат выполняется с помощью модуля zone_editor. После запуска одним кликом размечается верхняя точка запрещённой территории, вторым кликом размечается нижняя противоположная точка. После этого сохранение координат происходит нажатием "s", а выход без сохраниения нажатием "q".   
### Тестовое видео должно находится в директории media
